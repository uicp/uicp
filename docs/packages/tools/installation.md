# @uicp/tools Installation

Install and set up the UICP tools package for your backend/server.

## Installation

```bash
npm install @uicp/tools
```

## Requirements

- **Node.js**: 18 or higher
- **Environment**: Node.js (backend/server only)

## Package Information

| Property | Value |
|----------|-------|
| **Package Name** | `@uicp/tools` |
| **Size** | ~5KB gzipped |
| **Dependencies** | None |
| **License** | MIT |

## Verification

```typescript
import { getUIComponents, createUIComponent } from '@uicp/tools';

console.log('UICP Tools installed successfully');
```
